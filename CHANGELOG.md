# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-28

Initial daily-use release.

- Add single-file create, pull, update, delete, and sync commands.
- Add workspace init, team switching, info, import, pull, push, and delete commands.
- Add conflict protection through `hackmd_remote_hash`.
- Add `hackmd-cli` command templates for personal notes and team notes.
- Add offline Vimscript tests with a fake HackMD CLI.
- Add English and Chinese documentation.
